# Disor 1

أول إصدار من بوت Discord بيفهم كلامك العادي وينفذه باستخدام AI.

## إيه ده؟

بوت ديسكورد عادي بيستخدم أوامر محددة زي `!kick`، إنما Disor بيفهم اللغة الطبيعية ويحولها لأوامر تلقائياً.

مثال:
"اعملي روم سميه general-chat"
والبوت يفهم ويعمل الأمر المطلوب.

## إزاي شغال؟

النظام مبني على 3 موديلات:

1. **Router** - يحدد إذا كان المستخدم عايز يتكلم أو يعمل أمر
2. **Chat** - يرد على المستخدم عادي لو مش عايز أمر
3. **Action** - يحول الكلام لـ JSON عشان البوت ينفذه

## الأوامر المتاحة حالياً

- إنشاء رومات
- حذف رومات
- انشاء كاتجوري
- انشاء رتب
- تعديل رومات
- اعطاء رتب
